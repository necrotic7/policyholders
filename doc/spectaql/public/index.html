PolicyHolder

一個簡易的api server，用於管理保戶與保單的新增/修改/查詢功能。

使用以下工具進行開發：

  • typescript
    • ESModule
    • 使用 tsx
  • GraphQL
    • 使用 Apollo Server
    • schema 與 resolver 使用 type-graphql 撰寫
  • Koa
    • 作為 Server 框架
  • Oracle
    • 作為主要資料庫語言
API Endpoints
http://localhost:3000

Example

Query
query getPolicy(
  $policyHolderCode: Int,
  $policyID: Int
) {
  getPolicy(
    policyHolderCode: $policyHolderCode,
    policyID: $policyID
  ) {
    id
    description
    holder_name
    holder_code
    premium
    create_date
  }
}

    ]
  }
}

ID

Description

The ID scalar type represents a unique identifier, often used to refetch an object or as key for a cache. The ID type appears in a JSON response as a String; however, it is not intended to be human-readable. When expected as an input type, any string (such as "4") or integer (such as 4) input value will be accepted as an ID.
